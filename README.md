# CrazyFire80
CRAZY FIRE PART-1 for PC-8001

これは、1983年頃に開発されたPC-8001用ゲーム、CRAZY FIREのバイナリー及びソースコードです。

非商用に関してはOSSとして無償で利用自由です。改変も自由です。

商用利用に関してはTwitterの@KawamataAkiraまでお問い合わせください。

ここに掲載されたソースコードは手元にあるファイルを全て置いているだけなので、バイナリーを適切に生成できない可能性があることをお断りしておきます。また既に内容をかなり忘れているため、質問には対応できない可能性があります。

バイナリーの作成にはCP/M及びMACRO80が使用されていたものと思われます。

拡張子AZAのファイルはMINI-DOS用のAki's Zilog-mnemonic Assemblerのソースで、ほぼザイログニモニックのZ80アセンブラですが、16進数の表記として$を前置しても良かったり、特殊なマクロ命令が含まれる等の相違があります。
